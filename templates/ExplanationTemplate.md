# Características del Template:

## Tema oscuro:

    Fondo oscuro (#121212) y texto claro (#e0e0e0).
    Títulos en blanco (#ffffff) para mayor contraste.

## Secciones destacadas con bordes sutiles.

### Tipografía:

    Tamaño de letra un poco más pequeño (0.95rem para el cuerpo y 0.9rem para el código).
    Espaciado entre líneas mejorado (line-height: 1.6).

### Componentes:

    Listas numeradas y no numeradas con fondo oscuro.
    Código resaltado con fondo oscuro y texto claro.
    Tablas con estilo oscuro y bordes visibles.

### Flexibilidad:

    Puedes agregar o eliminar secciones según sea necesario.
    Ideal para documentación técnica, manuales o guías.
